Google Cloud CLI Essentials

1. Login to Google Cloud:
	gcloud auth login

2. Application Default Login (for local development):
	gcloud auth application-default login

3. Set Project ID:
	gcloud config set project <PROJECT_ID>

4. Impersonate a Service Account (for a single command):
	gcloud <COMMAND> --impersonate-service-account=<SERVICE_ACCOUNT_EMAIL>

5. Set Impersonation as Default (for all commands in session):
	gcloud config set auth/impersonate_service_account <SERVICE_ACCOUNT_EMAIL>
